# Contract and Payment System

Key considerations include:
● Implementing transactions and locks
● Handling race conditions
● Ensuring concurrency at scale
● Adhering to code style (SOLID principles and software design patterns)
● Writing efficient and optimized database queries
● Writing tests

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
